ChartLegend
===========

Legend addition for Chart.js.

This creates a legend for graphs created with Chart.js (http://chartjs.org).
Legend is pure HTML with some styling controlled by given options.

Based on https://github.com/bebraw/Chart.js.legend/blob/master/src/legend.js
and inspired by Chart.js itself.

Supports data with strokeColor arrays seen in some forks and the default 
string-valued property.

Implements two flavours for horizontal and vertical legends, see example.html.
